<h1 align="center">owl-twin</h1>
<p align="center">
  Digital twin pipelines for water system modelling and analysis
</p>

-----

Part of **[OpenWaterLab(OWL)](https://github.com/OpenWaterLab)** - free and open-source tools for water system modelling, simulation and analysis.

Created and maintained by: **[BIOMATH-UGent](https://github.com/UGentBiomath)**

**owl-twin**: A Python package providing reusable building blocks and automated pipelines for creating and maintaining digital twins of water systems.

Code of Conduct
-----

owl-twin is a community-driven free and open-source project developed by contributors to OpenWaterLab (OWL). It committed to providing a welcoming, inclusive, and collaborative environment for everyone involved in the project. Please follow guidelines provided in OpenWaterLab (OWL) [code of conduct](https://github.com/OpenWaterLab/owl-data?tab=coc-ov-file).
